# Домашнее задание: Список предложений (Listing)

Реализация компонента `Listing` для вывода списка товаров Etsy с использованием React.

## Особенности реализации:
* Названия товаров длиннее 50 символов обрезаются с добавлением `…`
* Корректное форматирование цен для USD ($), EUR (€) и GBP (£).
* Цветовая индикация остатков (зеленый/желтый/красный) в зависимости от количества товара.
* Исключены удаленные товары (без картинок или со статусом, отличным от 'active').
