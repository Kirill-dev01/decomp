# Домашнее задание: Декомпозиция

В данном проекте проведена декомпозиция сложного интерфейса (аналог главной страницы поисковика) на независимые и переиспользуемые React-компоненты.

## Особенности реализации:
* **Изоляция логики:** Созданы отдельные компоненты для списков новостей, поисковой строки, рекламного баннера и боковых виджетов.
* **Переиспользование (`props` и `children`):** Реализован универсальный компонент `Widget`, принимающий заголовок через пропсы и любое содержимое через `children`.
* **Документирование:** Компоненты описаны с использованием стандарта TSDoc.
* **TypeScript:** Код написан с использованием строгой типизации интерфейсов.
